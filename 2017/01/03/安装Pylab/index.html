Window下安装Pylab

2017-01-03

前一段时间需要用到Pylab，本来想着就是一个Python的模块安装应该还挺容易，后来才发现果然还是Too naive!

直接入题吧，我这里用的还是Python2.7。

Pylab依赖于SciPy、NumPy和Matplotlib三个软件包，所以首先需要安装这三个工具包：

NumPy:https://sourceforge.net/projects/numpy/files/NumPy/1.8.1/ 选择合适你的.exe直接下载安装，不需多说。

SciPy:https://sourceforge.net/projects/scipy/files/scipy/ 同样选择一个合适的版本直接下载安装。

Matplotlib:http://download.csdn.net/download/jakung/4434836 直接下载安装。

三个安装包安装完毕之后，打开python命令行：win+R 打开cmd命令行，输入python进入python环境（前提是Python.exe在系统环境变量里，具体设置百度之）：

输入命令：

import pylab

系统会提示错误，并指明你缺少某个模块。解决办法就是缺少什么模块就安装什么模块。

exit()

退出Python环境，用pip工具安装。

同样在cmd里输入pip即可查看pip命令（pip.exe同样要在系统环境变量里）：

例如，提示错误说缺少 pyparing组件，则到这里->http://www.lfd.uci.edu/~gohlke/pythonlibs/ 查找并下载相应组件（直接用ctrl+F查找很方便），找到 pyparsing-2.1.10-py2.py3-none-any.whl 文件后下载。 在cmd里输入：

pip install D:\安装pylab\pyparsing-2.1.10-py2.py3-none-any.whl

回车即可安装成功。注：命令里install后面是刚刚下载安装包的绝对路径，不方便输入，可把.whl文件直接拖入cmd窗口即可生成路径。

安装完一个组件就按照前面一步 import pylab一次，缺少什么就去查找然后下载安装。一个一个安装后，最后输入import pylab 没有任何错误提示，pylab就安装成功了。

过程参考网址：

http://jingyan.baidu.com/article/e3c78d644b21f23c4c85f5f8.html

http://blog.csdn.net/u012700322/article/details/50444856

小提示： 一些莫名奇怪的问题可以重启Python软件解决和重启电脑解决。重启大法好！！
